# Naka

This project is a personal project to me for learning Angular + Spring Boot + MySQL based on javainuse FullStack tutorial.

This project aims to be website who could replace this : https://docs.google.com/spreadsheets/d/1IZTdTsXEWMJ5_VREQm000eDkNY74sqTyMjRoxBNMnR0/edit#gid=834618439
used by the Youtuber Nakatine in his videos and for his followers.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`.

## Run Spring

Run "boot-naka" as Java Application. Listening to `http://localhost:8080/`. Database will automatically be created.

## Datas for database

Execute the script data-init-naka.sql to fill up database. Use `source data-init-naka.sql` in command line.

## Access database

Change application.properties in boot-naka according to your personnal SQL ( password & username ).
